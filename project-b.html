Project A: Low-poly Animals

Hi! I'm Bew, and to the left you can see my eagle and fox. The scene is currently set to load with the animals directly facing you. However, you can interact with them with the following controls:

  1. Drag anywhere within the canvas to rotate the animals around.
  2. Use the keyboard arrow keys to move the animals around on the canvas.
  3. Click anywhere in the canvas to randomly change the color of the animals.
  4. Press spacebar to play/pause all animations.
